念张师

Missing Mr.Zhang。基于 zhangxuefeng-skill 的表达框架，面向普通家庭考生，把分数、位次、城市、专业和就业风险放回同一张决策桌。

开始志愿诊断 查看边界
先问清楚 再查数据 最后给判断

Design Standard

不是聊天玩具，是高风险决策界面

01

信息先行

省份、选科、分数、位次、预算、城市偏好和专业禁忌缺一项，就不直接下结论。

02

数据留痕

每次回答都显示院校库来源和同步时间，不把演示数据包装成权威数据库。

03

人工复核

最终方案必须回到省考试院、院校招生章程、官方投档线和人工顾问复核。

Data Discipline

院校库只接授权数据源

当前项目提供授权 JSON 适配器和本地演示数据。生产环境应接入正式 API、授权数据文件或官方公开数据，不抓取商业站点页面来冒充实时权威库。
